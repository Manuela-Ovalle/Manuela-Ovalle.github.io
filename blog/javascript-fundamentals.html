What is control flow in Javascript?

Control flow is the direction that a computer follows when running code. Just like when we read English, the computer starts from left to right and from top to bottom.
In JavaScript we can use loops, conditionals and functions to change this order. To continue on my previous association of reading English, one could compare the change in flow to reading a newspaper spread, the reader’s gaze may not necessarily read from top to bottom but instead scan the headlines and bounce around the page.

What is The Dom and how do we interact with it?

The DOM, (document object model) is a tool that allows a developer to view code in several ways directly on the browser. This tool displays the html document as the nodes and objects allowing us to use javascript to interact with the page. One way that we can interact with the document is using the console.log command to run and debug the code. We can also use the querySelector command to select specific elements and manipulate them using other commands such as eventListener allow us to have popups or changes when a user of the site clicks or hovers over those elements.

What are arrays and objects? How do we access data frm them?

An array is a collection of data that is displayed using square brackets []. This collection is referred to as an index and each piece of data is numbered allowing the programmer to select and use individual pieces of data by referencing the index position of that chosen piece of data. We begin counting index positions with the number 0 not the number 1. So for example, in an array of 4 pieces of data the index positions would be [0,1,2,3].

Here is an example of what an array could look like:


let shoes = [nike, Adidas, vans, new balance]

index positions =[0 , 1 , 2 , 3 ]


Here I will choose vans which have the 2 index position.


let shoe = shoes[2];

The word object in javascript describes a standalone entity with its own properties and type. We can think of objects in code the way we think of objects in real life. Declaring objects is useful because we can assign particular properties of that object for example, if our object is a coffee, we can give it a price, a stock amount, the cost of making the coffee and so on.

Lets declare an object:

const cactus = {

type: ‘San Pedro’

height: ‘6 meters’

location: ‘Auckland Botanic Gardens’
}
