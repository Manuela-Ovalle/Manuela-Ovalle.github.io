Class attributes and ID attributes

What's the difference?

My page is full of div tags, p tags and h3 tags. I don't just have one set of each, I have many of them and often for a range of uses.

Using id attributes and class attributes for these tags has stopped me from getting lost in my code! Most importantly, these attributes give developers the ability to format these tags in different ways. (I must be honest though, at first, I didn't understand the difference between them and I would use them interchangeably lol.) However, they are not the same! Let's have a look at why they are different and how we can use them to make our code more organised and clear.

Class Attributes

H2 tags within the "school" class

As you can see in my doodle above, I have four h2 heading tags within my html file. To make sure I can keep their apparence consistant throughout my site, I've given them the same class "school".

HTML:

<h2 class="school">Home Page</h2>
<h2 class="school">Blog Page</h2>
<h2 class="school">Contact Page</h2>
<h2 class="school">About Page</h2>

CSS:

.school{
font-size:40px;
font-family: helvetica;
}

ID attributes

Id attributes are just as their name states, identification attributes. Like the doodle above, they give us a unique identifier, I think of them like using an ID card with a unique number for each individual. If you want your individual elements to have their own unique characteristics, using id tags can make this happen.

In our example of the doodles above, each of the h2 header tags have their own unique id attributes such as "home-page-header" or "blog-page-header". Which I can then use to make each of my headings have a unique color.

HTML:

<h2 id="home-header">Home Page</h2> <h2 id="blog-header">Blog Page</h2> <h2 id="contact-header">Contact Page</h2> <h2 id="about-header">About Page</h2>

CSS:

#home-header{
color: #95380c;
}
#blog-header{
color: #5656ff;
}
#contact-header{
color: #f656ff;
}
#about-header{
color: #bcf808;
}

Can class attributes and ID attributes be used on the same element?

Absolutely! As you can see in the doodle example, my 4 h2 tags have the "school" class attribute. This ensures that I can make them look identitcal. However, I have decided that I want each of these headings to have a different colour and using an ID attribute enables me to do that.

HTML:

<h2 id="home-header" class="school">Home Page</h2>
<h2 id="blog-header" class="school"> Blog Page</h2>
<h2 id="contact-header" class="school">Contact Page</h2>
<h2 id="about-header" class="school">About Page</h2>
